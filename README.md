# YTD
A simple YouTube video downloader using `pytubefix` for the back-end and `customtkinter` for the UI.

---

## Features:
1. A URL fetcher to make sure the video actually exists.
2. Download the video with different resolution options (from 144p up to whatever the video can go up to).
3. Internet speed indicator, progress bar with percentage, and estimated download time, all for the conveneince to your ADHD.
4. Cancel at anytime during the downloading process.
5. Open the folder where the video was downloaded using a simple button.

---

## Usage:
1. Clone repo:
    ```bash
    git clone 
    ```
2. Install requirements:
    ```bash
    pip install requirements.txt
    ```
3. run the app:
    ```bash
    python main.py
    ```
